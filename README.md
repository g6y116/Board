# Board 클론 코딩

## 배포

> http://54.180.205.236:8080

## 기능

- 회원가입
- 로그인/로그아웃
- 질문 CRUD
- 답변 CRUD
- 추천

## 구현

- 스프링 시큐리티 적용
- 질문 계시판 페이징
- 리스트에 답변/추천 수 표시
- 마크다운 형식으로 표시
- AWS를 활용하여 배포
